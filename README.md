# Lokesh Mohanty's Dotfiles

My configuration files for linux.

## Installation

    git clone https://github.com/lokesh1197/dotfiles.git
    cd dotfiles
    sh ./install.sh

## Necessary Applications
    - `git`
    - `neovim`
    - `mupdf`
    - `aria2`
    - `redshift`
    - `ntfs-3g`
    - `os-prober`
    - `haskell stack`
    - `mplayer`(video player)
    - `feh`(image viewer)
    - `krita`(image editor)
